This program prompts the user to enter input files of rational expressions
and it will calculate these mathematical expressions and print them onto 
an output file for you.

To compile the program, run the command:
make

To run the program, run the command:

./rational_math input1.txt output.txt

or you can run multiple files with...
./rational_math input1.txt input2.txt output.txt
